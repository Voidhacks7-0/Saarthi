COMPONENT DESCRIPTIONS:
=======================

1. FACULTY QUICK ACTIONS (faculty-quick-actions.tsx)
   - 3 main action buttons displayed prominently
   - Create Announcement: Navigate to announcement creation page
   - Create Event: Navigate to event creation page
   - Manage Groups: View and manage course discussion groups
   - Responsive grid (1 column mobile, 3 columns desktop)
   - Icons with gradient backgrounds for visual appeal

2. FACULTY ANNOUNCEMENTS (faculty-announcements.tsx)
   - Shows announcements created by the faculty member
   - Each announcement card displays:
     * Title
     * Short description
     * Audience (e.g., "CS101 - Section A", "All Classes")
     * Posted time (e.g., "2h ago")
     * Edit button (pencil icon)
     * Delete button (trash icon)
   - "View All" link to full announcements page
   - Responsive card layout

3. FACULTY EVENTS (faculty-events.tsx)
   - Displays upcoming events created by the faculty
   - Each event shows:
     * Event title
     * Date & time (with calendar icon)
     * Location (with map pin icon)
     * RSVP count (number of students who confirmed)
     * Edit button
     * Delete button
   - "View All" link to full events calendar

4. FACULTY COURSES (faculty-courses.tsx)
   - Lists all courses taught by the faculty member
   - Each course card includes:
     * Course name/code
     * "Open Forum" button - opens course discussion board
     * "Announce" button - quick link to create announcement for that course
   - Responsive grid layout (1-2 columns)
   - Shows "No courses assigned" if empty

5. FACULTY GROUPS & FORUMS (faculty-groups.tsx)
   - Shows course discussion groups/forums
   - Each group card displays:
     * Group name (e.g., "CS101 - Discussion Board")
     * Associated course name
     * Number of students in the group
     * Unread messages count
     * "Open Group" button to enter discussion
   - Faculty can view posts, reply to student questions
   - Faculty can pin important posts and delete inappropriate content
   - "View All" link to full groups page

6. FACULTY BOOKINGS APPROVAL (faculty-bookings.tsx)
   - ONLY visible if the faculty member is a Lab In-Charge
   - Shows pending booking requests from students
   - Each request displays:
     * Student name with avatar
     * Resource name (lab equipment or lab room)
     * Requested date & time
     * Purpose of booking
     * Approve button (green)
     * Reject button (red)
   - Status indicator showing number of pending requests
   - Hides automatically if faculty is not lab in-charge

7. FACULTY STATS (faculty-stats.tsx)
   - 4 quick stat cards showing:
     * Total announcements posted
     * Total events created
     * Courses managed (count)
     * Pending bookings (if lab in-charge)
   - Each card has icon, label, and value
   - Color-coded with gradient backgrounds
   - Responsive (1 column mobile, 2 columns tablet, 4 columns desktop)

RESPONSIVE DESIGN:
==================

Mobile (< 768px):
  - Sidebar collapses into hamburger menu
  - All sections stack vertically (1 column)
  - Quick actions: 1 column grid
  - Stats: 1-2 columns
  - Buttons take full width
  - Welcome banner hidden (space saving)
  - Optimized padding and spacing

Tablet (768px - 1024px):
  - Sidebar visible as sidebar
  - Main content in 2-column layout where applicable
  - Quick actions: 2-3 columns
  - Stats: 2 columns
  - Reduced font sizes for fit

Desktop (> 1024px):
  - Full 3-column layout (left + right sidebar)
  - Welcome banner visible
  - Quick actions: 3 columns
  - Stats: 4 columns
  - Full padding and spacing
  - Optimized for large screens
